위치 결정 방식

  • Normal Flow 정상적인 흐름 
    				div {
					position: static;
				}
  • Relative Position 상대 위치 
    div {
	position: relative;
}
  • Absolute Position 절대 위치 
    				div {
					position: absolute;
				}
  • Fixed Position 고정 위치 
    				div {
					position: fixed;
				}
  • Floating Element 부유요소 
    				div {
					float: left;
				}
